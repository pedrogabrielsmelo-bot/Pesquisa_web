# Pesquisa de Opinião - TudoWeb

## 📋 Descrição
Este projeto foi desenvolvido para realizar uma pesquisa de opinião com clientes da empresa **TudoWeb**, com o objetivo de avaliar o grau de satisfação no atendimento prestado.

O programa foi implementado em **Python** utilizando a estrutura de repetição `for` e estruturas de decisão (`if/elif`).

## 🚀 Funcionalidades
- Coleta de dados de 50 entrevistados (nome, idade e opinião).
- Opções de opinião:
  - 1: EXCELENTE
  - 2: BOM
  - 3: RUIM
- Contabiliza automaticamente:
  - Quantidade de respostas **EXCELENTE**
  - Quantidade de respostas **RUIM**

## 🛠️ Tecnologias utilizadas
- Python 3.x
